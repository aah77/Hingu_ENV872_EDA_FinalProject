# Hingu_ENV872_EDA_FinalProject
Repository for Aman Hingu's Final Project for ENV872

The title of this project is "Low Birth Weight Prevalence and Socioeconomic Factors in Chicago", by Aman Hingu. For this project, the dataset used was the Chicago Health Indicators dataset which has public health and socioeconomic indicator data for 77 community areas of the greater Chicago Metropolitan Area. The link to this dataset can be found below:
https://geodacenter.github.io/data-and-lab//healthindicators-variables/
